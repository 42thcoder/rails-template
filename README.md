# rails-template

A quick Ruby on Rails template for 80percent team

`$ bundle config mirror.https://rubygems.org https://gems.ruby-china.org`

`$ rails new myapp -m https://raw.github.com/80percent/rails-template/master/composer.rb`

## Tech stack

* Ruby on Rails 5
* bootstrap 3
* font-awesome
* figrao
* postgres
* slim
* high_voltage
* carriewave & upyun
* sidekiq
* kaminari
* mina
* puma
* lograge
